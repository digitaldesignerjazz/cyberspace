# Worlds

This folder contains world-building frameworks, setting documents, and examples for Nexus Cyberspace.

## Current Focus

- Cyberpunk megacities with decentralized infrastructure
- Fantasy realms connected through hidden mesh networks
- Hybrid worlds where magic and advanced technology coexist
- Post-collapse societies rebuilding through decentralized systems

## Guidelines

When contributing new worlds or expanding existing ones, please consider:
- How the world interacts with the underlying Nexus technical layers
- Opportunities for agent-driven narratives
- Consistent tone and internal logic

See the main `CONTRIBUTING.md` for more details.
